# Zum Thema Morsecode

*Eine antiquierte Form nautischer Kommunikation? Was kommt als nächstes? Wenigstens sind es echte Morsezeichen... Wenn du also aufpasst, kannst du zumindest noch etwas lernen.*

Interpretiere das Signal der blinkenden Lampe mithilfe
der Morsezeichen-Tabelle, um eines der Wörter aus der
Tabelle rechts zu erhalten.
Das Signal wiederholt sich in einer Schleife, mit einer langen Pause
zwischen den Wiederholungen.
Sobald das Wort identifiziert ist, stelle die zu dem Wort gehörende
Frequenz ein und drücke den "Senden" Knopf.

Morsealphabet
1. Eine kurzes Signal repräsentiert einen Punkt.

2. Ein langes Signal repräsentiert einen Strich.

3. Zwischen Buchstaben ist eine lange Pause.

4. Bevor das Wort wiederholt wird, gibt es eine sehr
     lange Pause.
